這個 repository 是我的學校作業與課程專案總整理，根目錄提供快速導航；每個子資料夾則放完整程式碼、實驗紀錄與專案 README。

## 使用方式

1. 先從上表點進對應資料夾。
2. 閱讀子專案 `README.md` 了解背景、環境需求與執行方式。
3. 若該專案含多子模組，優先查看子資料夾中的 `Makefile`、`Dockerfile` 或 `*.sh` 腳本。

## 備註

- 本 repo 以「課程學習紀錄」為主，不同專案使用的語言與建置流程彼此獨立。
- 若要重現結果，請依各專案 README 的相依套件與版本為準。
